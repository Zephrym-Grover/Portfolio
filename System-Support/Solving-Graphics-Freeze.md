# Solving a Graphics Freeze on a Laptop
### There are a lot of things that can cause a graphics freeze, particularly on a laptop. 

## Section 1.) Symptoms

- This is often described as the computer simply "Freezing up" on someone. If the computer does not recognize any inputs, including ctrl-alt-delete, mouse input, touchpad input, and keyboard input, a graphics freeze has likely occured
## Solving the issue in the immediate present

- Powering the computer off and then on again via the power button is often the only way to solve this if the computer does not respond to any input
- However, if the issue is caused by bad graphics drivers (One of the possible causes mentioned below), WinKey + ctrl + shift + B is a shortcut that can restart the graphics driver. The computer will beep, and the screen will flash. If it's just a graphics issue, this will solve the problem

## Possible causes and solutions

### Cause #1: Corrupted graphics drivers:

- While this issue can be solved in the immediate term through the method described above, that method is hardly a long-term fix for the issue. Thankfully, the actual issue does not take long to solve at all.
- Bad drivers are a common issue. Windows Update does not reliably bundle good GPU driver updates in their update package. Additionally, if you haven't manually updated your drivers recently, Windows will sneak a generic, stripped-down version of the driver in the background. This background driver can cause a bevy of issues, this being one of them.
- 1.) Go to the website of your graphics manufacturer (NVIDIA, AMD, Intel are the most common)
- 2.) Search for "Drivers"
- 3.) Find the graphics processing unit that your computer posesses (You can check what graphics card your laptop uses through this command)
- 4.) Search for that GPU on whichever website it is manufactured by (Some manufacturers, such as AMD, have an all-in-one package that will find the GPU of your computer after installation
- 5.) Download the graphics driver package
- 6.) Follow the installation wizard launched by the download
