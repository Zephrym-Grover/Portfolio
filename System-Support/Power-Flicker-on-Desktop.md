# Solving Power Flickers on Desktop
## There are a number of things that can cause power to flicker on a desktop PC

### Symptoms:

- Computer not powering on when it is supposed to
- Computer suddenly powering off (including all peripherals attached to PC)

### Cause #1.) Inadequate cooling
- Many PC's have features that automatically shut down the computer if internal components are too hot. This prevents damage to the computer's components via overheating.
- If overclocking is enabled, disable overclocking. Overclocking a computer can often cause it to overheat
- Check the fans. Ensure they are running smoothly with no abnormal noises. It should be nothing more than a minor, high-pitched hum. Any buzzing, whistling, or squeaking sound is cause for concern. If a fan is stopped, ensure that there is nothing restricting its movement, that it is not overly clogged with dust, and that the bearings are not stuck and in need of oil
- 
### Cause #2.) Virus

### Cause #3.) Airgap
- Ensure that the computer is fully plugged into the outlet. Additionally, ensure that any adapter used to plug in the computer is properly rated for that computer.
- Ensure that there is power on the breaker that the computer is being plugged into
- If the computer is plugged into the power bar, ensure that the switch on the end is toggled so that power is ENABLED to the strip  

### Cause #4.) Faulty connector

### Cause #5.) Faulty power supply
- NOTE: Power supplies contain capacitors that hold dangerous amounts of electricity, even when the computer is powered off. handling of these can result in SEVERE INJURY OR DEATH. Any investigation in the power supply should be performed with EXTREME caution by an experienced professional
